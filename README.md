# cs-learning

CS 学习笔记与代码归档。

## 目录约定

```
cs-learning/
├── algorithms/      # 算法与数据结构
├── systems/         # 操作系统、计算机网络、组成原理
├── languages/       # 编程语言练手（C / C++ / Python / Java / Rust …）
├── courses/         # 课程作业与 lab（按学期/课程分子目录）
├── projects/        # 小项目、玩具实现
└── notes/           # Markdown 笔记与读书摘录
```

需要哪个目录再建，不强求一次铺满。

## 提交习惯

- commit message 中英文皆可，能让未来的自己看懂即可
- 大文件（数据集、二进制、视频）走 [Git LFS](https://git-lfs.com/) 或 `.gitignore` 拦掉
- 涉及学术诚信的课程作业，**截止前**保持私有
